# Universal Unix Log Remover
<hr>
Copyright 2019 written by Misty;KISA
<hr/>

**automatic log remover for unix system**
```
This programe automatically deletes all of the contents of all log files
that are stored to unix or linux systems. you can easily remove with friendly ui.
```

## Requirement
- python 2.x

## Usage
```
$ python UULR.py
```
